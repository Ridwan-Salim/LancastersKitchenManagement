# Lancaster's Kitchen — Management System

A management platform designed for Lancaster's Restaurant to streamline operations, improve staff coordination, and provide data-driven insights.  
Developed for **IN2033 Team Project (2023/24)** by **Group 9 — 9 Rings**.

---

## Features

### Staff & Schedule Management
- Secure login for staff and managers
- View employee shifts
- Create and edit weekly staff schedules
- Manage and approve holiday requests

### Restaurant Operations
- Send ingredient and supply orders via API
- Edit and manage restaurant menu items
- Monitor wine cellar inventory and stock

### Insights & Forecasting
- Booking prediction system for demand forecasting
- Restaurant statistics dashboard with key performance metrics
